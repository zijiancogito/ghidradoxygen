21 // Structures for splitting big varnodes carrying smaller logical
22 // variables. Given a root within the syntax tree and dimensions
23 // of a logical variable, this class traces the flow of this
24 // logical variable through its containing varnodes. It then
25 // creates a subgraph of this flow, where there is a correspondence
26 // between nodes in the subgraph and nodes in the original graph
27 // containing the logical variable. When doReplacement is called,
28 // this subgraph is duplicated as a new separate piece within the
29 // syntax tree. Ops are replaced to reflect the manipulation of
30 // of the logical variable, rather than the containing variable.
31 // Operations in the original graph which pluck out the logical
32 // variable from the containing variable, are replaced with copies
33 // from the corresponding node in the new section of the graph,
34 // which frequently causes the operations on the original container
35 // varnodes to becomes deadcode.
