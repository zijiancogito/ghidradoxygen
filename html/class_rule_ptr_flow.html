Detailed Description

Mark Varnode and PcodeOp objects that are carrying or operating on pointers.

This is used on architectures where the data-flow for pointer values needs to be truncated. This marks the places where the truncation needs to happen. Then the SubvariableFlow actions do the actual truncation.
