Detailed Description

A container for Varnode objects from a specific function.

The API allows the creation, deletion, search, and iteration of Varnode objects from one function. The class maintains two ordering for efficiency:

  • Sorting based on storage location (loc)
  • Sorting based on point of definition (def) The class maintains a last offset counter for allocation temporary Varnode objects in the unique space. Constants are created by passing a constant address to the create() method.
