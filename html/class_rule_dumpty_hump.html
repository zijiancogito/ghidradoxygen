Detailed Description

Simplify join and break apart: sub( concat(V,W), c) => sub(W,c)

Depending on c, there are other variants:

  • sub( concat(V,W), 0) => W
  • sub( concat(V,W), c) => V
  • sub( concat(V,W), c) => sub(V,c)
